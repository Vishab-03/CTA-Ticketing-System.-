## Description
The CTA Ticketing System is a Python program that calculates ticket fares based on the starting and destination zones entered by the user.  
The program uses functions, constants, and input validation to ensure accurate results and a user-friendly experience.

This project was created as part of a programming assignment and demonstrates Merit-level coding practices.

---

## Features
- Calculates ticket fare using zone-based pricing
- Uses constants for base fare and zone rate
- Validates user input (zones must be between 1 and 5)
- Handles same-zone travel
- Allows multiple fare calculations in one run
- Displays formatted ticket price
- Simple command-line interface
