## Purpose
The CTA Ticketing System is a Python-based command-line application designed to calculate ticket fares based on travel zones.  
It provides a simple, user-friendly interface, ensures input validation, and allows multiple fare calculations in a single session.  
This program demonstrates Merit-level programming skills, including modular functions, constants, validation, and structured code.

---

## Features
- Calculates fare based on zone difference  
- Validates input (zones must be between 1 and 5)  
- Handles same-zone travel  
- Allows repeated fare calculations without restarting  
- Displays formatted ticket price  
- Simple and clear command-line interface
